    • Authority

    Authority helps you authorize actions in your Rails app. It's ORM-neutral and has very little fancy syntax; just group your models under one or more Authorizer classes and write plain Ruby methods on them.

    Usage

            
          class User
            include Authority::UserAbilities
          end

          class Post < ActiveRecord::Base
            include Authority::Abilities
          end

          class Comment < ActiveRecord::Base
            include Authority::Abilities
            self.authorizer_name = 'PostAuthorizer'
          end

          class PostAuthorizer < ApplicationAuthorizer
            def updatable_by?(user)
              user == resource.owner || user.admin?
            end
          end

          # Pretend we initialized these objects. ;)
          regular_user.can_update?(@own_post)    # => true
          regular_user.can_update?(@own_comment) # => true
          regular_user.can_update?(@my_post)     # => false
          regular_user.can_update?(@my_comment)  # => false
          admin_user.can_update?(@my_post)       # => true
          admin_user.can_update?(@my_comment)    # => true
